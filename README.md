## Getting Started

```sh
$ git clone https://github.com/PradyumnaKrishna/Django-Vue-Form
$ cd Django-Vue-Form/
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ cd userform
$ python manage.py migrate
$ python manage.py runserver
```
Application is started at 
```sh
127.0.0.1:8000
```

## Deploy on Heroku

- After Deploy to heroku using favorable method type:

```
$ heroku run -a <YOUR APP NAME> python userform/manage.py migrate
```
